# 08. KPI Design — KGIから逆算したKPI設計

作成日: 2026-07-07

---

## KGI(最終目標)

**日本一継続率が高いAI占いサービスになる**(継続率・課金率・口コミ量の3軸で業界トップを目指す)

## KPIツリー(KGIからの逆算)

```
KGI: 日本一の継続率
  │
  ├─ 継続率KPI
  │   ├─ Day1継続率(目標: 業界平均超の40%以上 ※業界平均は約29%)
  │   ├─ Day7継続率(目標: 25%以上)
  │   ├─ Day30継続率(目標: 15%以上、SELFアプリの50%を中長期目標に)
  │   └─ 週内3回以上起動ユーザー率(北極星指標)
  │
  ├─ 課金率KPI
  │   ├─ 無料→有料転換率(CVR)
  │   ├─ 有料会員の継続課金率(チャーン率の逆数)
  │   └─ LTV(顧客生涯価値)
  │
  ├─ 口コミKPI
  │   ├─ SNSシェア率(診断完了者のうちシェアした割合)
  │   ├─ バイラル係数(K-factor)
  │   └─ 指名検索数(「糸町の少年」検索ボリューム)
  │
  └─ 獲得KPI
      ├─ PV / UU
      ├─ チャネル別流入数(SEO/SNS/紹介/直接)
      └─ CAC(顧客獲得コスト)
```

## 指標一覧・定義

| 指標 | 定義 | 90日後の目標(仮) |
|---|---|---|
| PV | ページビュー数 | 月間10万PV |
| UU | ユニークユーザー数 | 月間2万UU |
| 登録率 | サイト訪問者のうち会員登録した割合 | 8% |
| Day1継続率 | 登録翌日に再訪したユーザー割合 | 40% |
| Day7継続率 | 登録7日後に再訪したユーザー割合 | 25% |
| Day30継続率 | 登録30日後に再訪したユーザー割合 | 15%(中長期30%) |
| CVR(無料→有料) | 無料会員のうちサブスク登録した割合 | 5% |
| チャーン率 | 月間解約率 | 8%以下 |
| LTV | 1ユーザーあたりの生涯収益 | 算出開始(初期はデータ収集フェーズ) |
| CAC | 1ユーザー獲得あたりのコスト | オーガニック中心のため低水準維持 |
| 紹介率 | 紹介経由の新規登録割合 | 10% |
| SNS流入 | SNS経由セッション数/月 | 3,000(Day30時点)→15,000(Day90) |
| SEO流入 | 検索経由セッション数/月 | 500(Day30)→10,000(Day90) |
| SNSシェア率 | 診断完了者のうちシェアボタン押下率 | 15% |

## KPI測定の実装要件(マーケティング部門から開発部門への申し送り)

以下はKPI計測のために必要な計測基盤。既存の`AnalyticsEvent`/`ExperimentAssignment`モデルが実装済みのため、イベント種別の追加で対応可能:

- 診断完了イベント(カテゴリ別)
- シェアボタン押下イベント(媒体別)
- 紹介コード経由登録イベント
- SNS流入元の計測(UTMパラメータ運用ルール策定)

詳細はIssue化(`10_Backlog.md`参照)。

## レポーティング頻度

- 週次: 獲得系KPI(PV/UU/流入チャネル別)
- 月次: 継続率・課金率・LTV
- 四半期: KGI全体の進捗レビュー
