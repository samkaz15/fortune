## 2. 柱①: 診断バズコンテンツ

### 企画A:「5つの四柱推命タイプ診断」

```
【フォーマット】TikTok / IG Reels（15-30秒）
【構成】
0-3秒: 「これ当たりすぎて怖いって話題の診断」（フック）
3-20秒: 5タイプを高速紹介（木・火・土・金・水）
20-30秒: 「あなたはどれ? コメントで教えて🌙」（CTA）

【テロップ例】
木のタイプ → 成長し続ける挑戦者
火のタイプ → 情熱で人を照らすリーダー
土のタイプ → みんなの支えになる安定型
金のタイプ → 鋭い分析力の戦略家
水のタイプ → 柔軟に流れる包容の人

【BGM】トレンド曲（月次で差し替え）
【狙い】自分のタイプ確認欲 → シェア → プロフィールのApp導線
```

### 企画B:「名前でわかる隠れた才能」（姓名判断）

```
【フォーマット】TikTok（20-40秒）
【構成】
- 「信じるか信じないかはあなた次第🌙」
- 名前の画数から才能を提示（バーナム効果を活用）
- 「詳しくはプロフから」

【注意】GPT3の禁止事項準拠（断定・不幸予言NG）
```

## 3. 柱②: 神社参詣コンテンツ（差別化の核）

### 企画C:「ツクヨミと巡る開運神社」シリーズ

```
【フォーマット】YouTube（5-10分）+ TikTok切り抜き（60秒）
【構成】
- オープニング: 神社の風景 + ASMR環境音（鳥・風・砂利音）
- 本編: CEO による参詣ガイド + 神社の歴史
- 占い要素:「このタイプの人にぴったりの神社」分析
- クロージング:「あなたも行ってみて。運気カレンダーで吉日をチェック🌙」

【差別化】
CEO一次情報 = 他社が模倣できないUnique Content（GM5準拠）
```
