# 07. Influencer Strategy — インフルエンサー戦略

作成日: 2026-07-07

---

## 1. 誰に依頼するべきか

2026年調査(占いコンパス記事等)より、占いジャンルで実際に支持されているインフルエンサーの型:

| 型 | 特徴 | 起用の狙い |
|---|---|---|
| **占星術・タロット系インフルエンサー** | しいたけ.型(詩的な語り口)、真木あかり型(雑誌連動) | ブランド世界観との親和性訴求 |
| **ライフスタイル系マイクロインフルエンサー** | 「丁寧な暮らし」「自己分析」文脈の発信者 | ターゲット層(20〜30代)への直接リーチ |
| **恋愛・キャリア系インフルエンサー** | 恋愛相談/キャリア相談を発信する層 | `/love` `/work`診断との直接連携 |

## 2. フォロワー帯の選定方針

- **マイクロインフルエンサー(1万〜10万フォロワー)を優先**。理由: エンゲージメント率が高く、費用対効果が良い。2026年のTikTok調査でも「フォロワー数よりエンゲージメント率・閲覧時間が重要」という傾向が明確化している
- メガインフルエンサー(50万超)は予算規模が合ってから検討(Cランク施策)

## 3. ジャンル

占い・スピリチュアル/ライフスタイル・自己啓発/恋愛相談/キャリア・転職 の4ジャンルから、各1〜2名を目安に試験起用する。

## 4. 費用感(目安・市場相場)

| フォロワー帯 | 想定費用(1投稿) | 備考 |
|---|---|---|
| 1〜3万人 | 3〜10万円 | ギフティング(無料利用提供)のみでも受諾されるケースあり |
| 3〜10万人 | 10〜30万円 | |
| 10万人〜 | 30万円〜 | 初期はここまでは手を出さない方針 |

**まずはギフティング(サービス無料提供+成果報酬)から始め、効果検証後に費用発生型へ移行する**段階的アプローチを推奨。

## 5. 期待効果

- フォロワーへの直接的な認知獲得
- インフルエンサー自身の「本音レビュー」によるUGC/信頼性訴求
- コラボ投稿のリポスト・引用によるSNSアルゴリズム上の露出増加

## 6. 依頼テンプレート(DM/メール文面)

```

はじめまして。AI占いサービス「糸町の少年」を運営しております〇〇と申します。
(お名前)様の発信をいつも拝見しており、占い・自己分析への向き合い方に
深く共感し、ご連絡させていただきました。

弊社サービスは、四柱推命・算命学などの東洋占術をベースに、
「今日の運勢」「相性診断」「仕事の適職診断」をAIキャラクター
「糸町の少年」と一緒に楽しめるサービスです。

もしご興味をお持ちいただけましたら、まずは無料でサービスを
お試しいただき、忌憚のないご感想をお聞かせいただけますと幸いです。
(ご感想をSNSでご紹介いただけるようでしたら、別途謝礼についてもご相談させてください)

お忙しいところ恐れ入りますが、ご検討いただけますと幸いです。
```

## 7. 選定・依頼フロー

1. 候補リストアップ(各ジャンル3〜5名)
2. ギフティング打診(上記テンプレート使用)
3. 反応があった相手とサービス無料提供+投稿依頼
4. 効果測定(流入数・エンゲージメント)
5. 成果が出た相手とは継続契約(費用発生型)へ移行
