## 2. 記事数・カテゴリ設計

| カテゴリ | 記事数目標(90日) | クラスター構造 |
|---|---|---|
| 診断コラム(自分/恋愛/仕事) | 15本 | 各診断ページへの内部リンク元 |
| 開運・運気コラム | 20本 | ピラー1本+クラスター記事群 |
| 神社・参詣コラム | 15本 | CEO一次情報の参詣レビューと連動 |
| 占術解説(四柱推命/算命学とは) | 10本 | EEAT訴求(専門性の証明) |
| FAQ | 1ページ(20問程度) | 全カテゴリから内部リンク |

## 3. コンテンツクラスター構造(ピラー+クラスター戦略)

```
ピラーページ: 「占いで人生の決断を後悔しないための完全ガイド」
├─ クラスター: 四柱推命とは(専門解説)→ /self へCTA
├─ クラスター: 相性診断の仕組み → /love へCTA
├─ クラスター: 適職診断の考え方 → /work へCTA
├─ クラスター: 開運日カレンダーの使い方 → /calendar へCTA
└─ クラスター: 参詣の正しい作法 → /shrines へCTA
```

## 4. 内部リンク設計

- 全コラム記事の末尾に「あなたの場合はどうなる? 無料で診断する」CTAを統一配置(既存の`GlassMosaic`/`FloatingCTA`と同じUXパターンをブログ記事にも適用)
- ピラーページ→クラスター記事→診断ページの3階層リンク構造
- 診断結果ページ(`/result/[id]`)からもコラムへの「次に読むと良い記事」リンクを設置(現状は`recommendNextCategory`で次の診断のみ推薦しているため、記事推薦も追加候補)

## 6. E-E-A-T対策(Experience, Expertise, Authoritativeness, Trustworthiness)

占いというYMYL(Your Money or Your Life)隣接領域では、Googleの評価基準上E-E-A-Tが特に重要。

- **Experience(経験)**: CEOの「参詣レビュー」(一次情報)を神社コンテンツに継続掲載 — 実装済みの`ShrineReview`モデルを活用
- **Expertise(専門性)**: 占術解説記事に「どの流派・どういうロジックに基づくか」を明記(占術の内訳非開示原則とは別に、**解説記事では占術体系そのものを説明することは許容範囲**。ユーザー向けチャット/診断結果に星名等を出さない原則と、教育的なブログ記事でその占術を解説することは別軸)
- **Authoritativeness(権威性)**: 運営会社情報(株式会社Viwe Point、所在地)を全記事フッターに明記(既に特商法ページで整備済みの情報を活用)
- **Trustworthiness(信頼性)**: プライバシーポリシー・データ取扱いの明記(2026年の調査で「運営会社の所在地」「データ第三者提供先の明記」が信頼構築に直結すると判明。既に整備済みの内容をSEO記事側にも露出させる)
