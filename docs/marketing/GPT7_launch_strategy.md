## 2. Stage 1: ソフトローンチ施策

### 目的とターゲット

```
目的: 「熱心なコアファン」の獲得と品質検証
規模: 1,000〜5,000人（招待制）
KPI:  NPS 40+、7日継続率 45%、重大バグ0件
```

### 施策

#### ① クローズドβ招待キャンペーン

```
- CEO のSNS・知人ネットワークから招待
- 「先行体験メンバー」の特別感を演出
- フィードバックで App内クレジット付与
- 招待コードによる紹介制（バイラルの種）
```

#### ② コアファン・コミュニティ形成

```
- 招待ユーザー限定のコミュニティ（Discord/LINEオープンチャット）
- CEO が直接参加し「占術の裏話」を共有
- ユーザーの声を Phase2 開発に反映（共創感）
```

#### ③ 神社参詣モニター企画（差別化の実証）

```
- β ユーザーに「推奨神社参詣」を体験してもらう
- 参詣レポートを UGC として蓄積
- GPT6 の「参詣チャレンジ」の先行テスト
```

## 3. Stage 2: 正式ローンチ施策

### 目的とターゲット

```
目的: ユーザー基盤の拡大 + 話題化
規模: 初月10万DL、3ヶ月で50万DL
KPI:  MAU 30万、課金導入率 20%（初期）→30%
```

### 施策（マルチチャネル同時展開）

#### ① SNSバイラル（GPT6連動）

```
- ローンチ告知の「診断バズ」動画を集中投下
- インフルエンサー10社同時投稿（占い系5+観光系5）
- #fortune参詣チャレンジ の本格始動
- 期待: ローンチ月に1000万リーチ
```

#### ② PR・メディア露出

```
- プレスリリース配信（PR TIMES等）
  切り口:「AIと神社をつなぐ新しい占い」
        「東洋占術3種を統合した本格AI占い」
- Web メディア寄稿（占い・ライフスタイル系）
- CEO インタビュー（占術師としての権威性訴求）
```

#### ③ ローンチキャンペーン

```
- 「ローンチ記念: 詳細占い7日間無料」
- 「友達招待で両者にクレジット」（紹介制度）
- 「初回オークション」を目玉イベント化（GM4連動）
```

#### ④ App Store 最適化（ASO）

```
- キーワード最適化（「占い」「四柱推命」「神社」）
- スクリーンショット・プレビュー動画の作り込み
- ローンチ時のレビュー獲得施策（β ユーザーに依頼）
- Featured 掲載を狙う（Apple/Google への申請）
```

## 5. 集客チャネル別 優先度・予算配分

GM8 の CAC 分析を反映。

| チャネル | CAC | 優先度 | Stage2予算配分 |
|---|---|---|---|
| SNSオーガニック | $0.5-1 | 最高 | 30% |
| インフルエンサー | $1-2 | 高 | 25% |
| PR・メディア | 低（換算難） | 高 | 15% |
| Facebook/Google広告 | $1-1.5 | 中 | 20% |
| ASO | $0.3 | 中 | 10% |

```
【方針】GM8準拠でオーガニック・口コミを主軸に。
有料広告は「勝ちパターン検証後」に段階投下（CAC暴騰を回避）。
```

## 6. ローンチ・ファネル設計

```
認知（SNS/PR）
    ↓ 「当たりすぎ診断」でフック
興味（プロフ→App Store）
    ↓ ASO最適化されたストアページ
DL・登録（ログインなし試し占い）
    ↓ GM3準拠の低摩擦オンボーディング
初回体験（毎日の運勢 + ツクヨミ）
    ↓ 「次の一歩」で価値実感（GPT4）
継続（通知 + 神社連携 + コミュニティ）
    ↓ 45%継続率を狙う
課金（詳細占い→サブスク→オークション）
    ↓ 自然な有料転換
紹介（友達招待 + SNSシェア）
    → バイラルループ
```

## 8. ローンチ・リスクと対策

| リスク | 影響 | 対策 |
|---|---|---|
| **サーバー負荷（急激な流入）** | 高 | Stage段階化 + CL29負荷試験（Phase3前倒し検討） |
| **重大バグの露出** | 高 | ソフトローンチで事前検証（CL13結合テスト） |
| **炎上（占いの倫理批判）** | 中 | GPT3の免責姿勢 + 法務確認 + 慎重な表現 |
| **話題が広がらない** | 中 | インフルエンサー複数同時投下でブースト |
| **課金への反発** | 中 | 「無料で十分価値」→自然転換（押し付けない） |
| **競合の妨害的追従** | 中 | 神社連携の独自性で差別化維持（GM5） |
