# 06. Content Strategy — コンテンツ戦略

作成日: 2026-07-07

SEO(`03_SEO.md`)・SNS(`04_SNS.md`)を横断する「コンテンツの作り方」を定義する。

---

## 1. コンテンツの3階層

```
① 診断コンテンツ(プロダクト本体) — self/love/work/report
② コラム・記事(SEO資産) — ブログ/note
③ SNS投稿(拡散装置) — TikTok/Instagram/X等
```

①が核、②③は①への導線として機能する。**すべてのコンテンツは最終的に診断への入口に接続する**設計を徹底する(既存のCVR設計思想=サブスク導線最優先、を継承)。

## 2. コラム・ブログのカテゴリ設計

| カテゴリ | 目的 | 更新頻度 | 誘導先 |
|---|---|---|---|
| 診断解説コラム | SEO+信頼性(EEAT) | 週2〜3本 | self/love/work |
| 開運・運気コラム | SEOロングテール獲得 | 週2本 | calendar/report |
| 神社レビュー | 差別化(競合にない一次情報) | 週1〜2本 | shrines |
| 占術解説(教育コンテンツ) | 専門性の証明 | 月2本 | ブランド全体 |
| ユーザーストーリー(体験談) | 社会的証明・UGC活用 | 月2本 | 全体 |

## 3. 占いコラムの型(テンプレート)

1. **導入**: 読者の悩み・疑問に共感する導入(不安煽りではなく共感)
2. **本編**: 占術的な背景を分かりやすく解説(専門用語は噛み砕く)
3. **実践**: 「あなたの場合はどうなるか、無料で診断してみましょう」というCTA
4. **関連コンテンツ**: 内部リンク3〜5本

## 4. 診断コンテンツの拡張案(将来のプロダクト機能候補)

現状の3診断(自分/恋愛/仕事)に加え、コンテンツマーケティング観点で需要が見込める拡張:

- 金運診断
- 相性診断(友人関係版、恋愛以外の人間関係)
- 今年の運勢(年間サマリー、Marketing観点でも「今年」の期間タブ拡張と相性が良い)

これらはプロダクト機能追加であり、マーケティング部門の管轄外(別途プロダクト側で検討)だが、コンテンツ戦略上の需要データとして記録しておく。

## 5. UGC(ユーザー生成コンテンツ)の活用

- ユーザーの診断結果シェア投稿を(許可を得た上で)公式SNSでリポスト・紹介
- 「みんなの相性診断結果」等、匿名集計データを使ったコンテンツ化(個人情報は含めない)

## 6. コンテンツカレンダーの運用

季節イベント(初詣、七夕、大晦日等)に合わせた特集コンテンツを`09_Roadmap.md`のロードマップと連動して計画する。
