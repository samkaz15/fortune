# CVR設計仕様: 結果画面・課金導線・登録後演出（2026-07-05）

**ソース**: CEO提供の市場調査（上位5サイトのレイアウト・課金・決済・演出分析）
**方針**: 構造（情報ギャップ・数字・即時性）は全面採用。**文言はブランドに翻訳**して採用。

---

## 0. ブランド整合の判断（重要）

調査の「キラー文言」は不安を煽る系（暴露・執着・問い詰める等）が中心だが、
これはCEO_STRAT「常にポジティブ」/LANGUAGE_DESIGNと衝突する。
→ **構造は踏襲し、言葉は糸町の少年トーンに翻訳する**。

| 調査の型 | 糸町の少年への翻訳例 |
|---|---|
| 「あの人の3つの本音」 | 「太郎さんの、いま言葉にしていない3つの気持ち」 |
| 「モザイクの向こう側を今すぐ問い詰める」 | 「この先は、僕から直接話しますね（残り 5/5）」 |
| 「〇月〇日の結末」 | 「動くのに一番いい日、見えてます」 |

数字（3つ/〇月〇日/残り回数）と情報ギャップ（寸止め）は維持する。
カウントダウンタイマー（30分限定等）は損失回避の煽りが強く、ブランド毀損リスクがあるため
**初期は非採用。採否はCEO判断待ち**（採用する場合も「今日の流れは今日まで」等の自然な期限表現に翻訳）。

## 2. 結果画面（1st Output）の必勝ワイヤー

上から: ①スコア＋チャート（一瞥で分かる可視化）→ ②無料で読ませる「本質」（約200字×2）→
③**核心の直前でグラデーションモザイク（CSS blur）による情報遮断** → ④CTA（残り回数表示付き）。
「全部読ませて終わる」は禁止。物足りなさを残した状態で5回チャット/課金導線へ合流させる。

## 5. 登録/決済後「3秒ドラマチック・ローディング」

旧仕様の1.5秒を**3秒・3段階文言**に更新（本ドキュメントが正）。

| 時刻 | 演出 |
|---|---|
| 0.0s | ボタン押下、裏でDB処理/AI生成開始 |
| 0.5s | 暗転→「糸をたどっています、、」 |
| 1.5s | 「今日の流れと重ねています（65%）」 |
| 2.5s | 「見えました。」＋光のエフェクト |
| 3.0s | 結果画面がフェードイン（核心はモザイク状態） |

目的: バックエンド遅延の隠蔽＋目標勾配効果による期待感の最大化。
文言はスキャン対象を具体的に（%表示あり）。ただし「LINEデータを同期中」等の
**事実でない演出文言は使わない**（実際に参照しているもの＝生年月日・会話ログ・今日の流れ、のみ言及可）。
