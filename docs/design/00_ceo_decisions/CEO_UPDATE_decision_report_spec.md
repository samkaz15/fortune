## 外部環境の翻訳ルール（重要）

「今日は低気圧なので」という気象説明は**禁止**。
天気・気圧は必ず人間行動キーワードへ翻訳する。
例: 低気圧 → 「疲労」「判断を急ぎやすい空気」「周囲も余裕がなくなりやすい」
