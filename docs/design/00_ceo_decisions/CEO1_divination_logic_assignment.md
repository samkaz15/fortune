## 決定事項：相談カテゴリ別の占術固定割当

ブレンド(複数占術のスコアを合成する方式)ではなく、**カテゴリごとに使用する占術を決め打ちする**。

| 相談カテゴリ | 使用する占術 | 対応する`ConsultCategory` |
|---|---|---|
| ビジネス・お金の相談 | 算命学 | `BUSINESS` |
| 自己分析・運気の良し悪し | ホロスコープ + 四柱推命 | `SELF`, `TODAY` |
| 人間関係・相性 | 姓名判断 | `RELATIONSHIP`, `COMPATIBILITY` |

## 実装方針

- `src/lib/fortune-engine/index.ts` の `generateFortune()` を、全占術を常に計算する方式から、
  **category に応じて必要な占術のみ計算する方式**に変更する。
- ユーザー向けにはCEO_STRAT決定の通り「複数のいい所を組み合わせている」とだけ伝え、
  実際にどの占術をどのカテゴリで使っているかの内訳は開示しない
  (システムプロンプト側で内部ロジックへの言及を禁止する)。
- 各占術ロジック自体(`seimei.ts` / `sanmei.ts` / `shichu.ts` / `horoscope.ts`)の
  計算精度(画数辞書・暦計算等)は暫定実装のまま。流派の指定は次回監修事項として残る。
