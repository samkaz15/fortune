## 質問カテゴリ → エンジンの対応（相談チャット）

| 質問の種類 | 使用エンジン |
|---|---|
| ビジネス関連の質問 | **算命学エンジン** |
| 相性占い関連の質問 | **姓名判断エンジン** |
| 今日の占いの質問 | **ホロスコープ + 四柱推命エンジン** |

- 恋愛・相性はUI上、専用ボックス（恋愛タブ）内で完結して質問できるようにし、
  **エンジンのスイッチコストが発生しない設計**とする（タブ＝エンジンの暗黙ルーティング）。
- エンジン名・占術名はユーザーには**一切開示しない**。すべて「錦糸町の少年式の占い方法」。

## 日次意思決定レポートの項目別エンジン（v3レビューで確定）

| レポート項目 | エンジン |
|---|---|
| ヘッダー・総合トーン / SUMMARY | ホロスコープ + 四柱推命 |
| 注意3項目の1つ目（その人に今日ネガティブが起こる可能性） | 四柱推命 |
| 注意3項目の2つ目（大衆的によく起こる悩み） | ホロスコープ |
| 注意3項目の3つ目（将来の星の元から気をつけるアドバイス） | 算命学 |
| アドバイス（転職などキャリア文脈） | 算命学 + 四柱推命 |

## 断言ルール（フィードバックの感情設計）

- 通常日: ポジティブ要素とネガティブ要素を組み合わせ、**感情を50%に是正**する
  （「今日はいい日ではあるのですが、注意は必要です」型）
- **例外**: 四柱推命の暦から見て「かなりいい日」は是正せず**いい日と断言**。
  同様に明確に悪い日は悪い日と率直に伝える（ただし必ず対処とセット、最後は前向きに着地）
