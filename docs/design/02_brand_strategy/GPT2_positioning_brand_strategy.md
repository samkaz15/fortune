
## 2. ブランドの3本柱（Brand Pillars）

GM5 の差別化軸を、顧客が体感できる「約束」に変換。

### Pillar 1: 信頼できる占術 ——「本物の知恵」

```
【顧客への約束】
「なんとなく」ではなく、3000年の東洋占術に基づいた、
あなただけの本格的な分析を。

【裏付け】
- 四柱推命 × 算命学 × 姓名判断の統合分析
- CEO（占術監修者）による品質保証
- 各占術の理論・根拠の可視化

【競合との差】
LINE占い/Yahoo!占い（浅い星座占い）に対する"深さ"
```

### Pillar 2: いつもそばにいるAI ——「続く関係」

```
【顧客への約束】
迷った夜も、嬉しい朝も。
ツクヨミが24時間、あなたに寄り添います。

【裏付け】
- キャラクター「ツクヨミ」による一貫した人格
- AIチャット相談（24時間）
- 毎日の運勢 + 具体的な"次の一歩"

【競合との差】
単発で終わる診断に対する"継続する伴走"
```

### Pillar 3: 実世界へつなぐ ——「行動できる占い」

```
【顧客への約束】
占いを"知る"だけで終わらせない。
あなたにぴったりの神社へ、実際の一歩を。

【裏付け】
- あなたの運気に合った神社の推奨
- CEO の参詣レビュー（一次情報）
- 参詣チャレンジ・運気カレンダー

【競合との差】
完全デジタル完結の他社に対する"リアルとの融合"（唯一無二）
```

## 3. ブランドパーソナリティ & トーン

### ブランドパーソナリティ（GPT1 と連動）

```
Sage（賢者）× Caregiver（世話役）のハイブリッド

- Sage: 深い知恵、信頼、本物志向
- Caregiver: 寄り添い、安心、無条件の味方

→ 「賢くて優しい、年上の親友」
```

### トーン・オブ・ボイス（4原則）

| 原則 | Do | Don't |
|---|---|---|
| **温かい（Warm）** | 「大丈夫、あなたなら」 | 冷たい分析口調 |
| **誠実（Honest）** | 根拠を示す | 大げさに煽る |
| **前向き（Uplifting）** | 必ず光を示す | 不安で終わらせる |
| **静か（Calm）** | 余白を残す | 押し付け・命令 |

## 6. 競合ポジショニングマップ

### 2軸マッピング

```
                    深い・本格的（占術の深さ）
                            ↑
                            │
          占いの館 ●        │        ● fortune（狙う位置）
        （高価格・対面）      │      （AI × 本格占術 × 神社）
                            │
   完全デジタル ───────────┼─────────── リアル融合
    （手軽・オンライン）      │       （実世界と接続）
                            │
        LINE占い ●          │
        ウラナ ●            │
      （手軽・浅い）          │
                            │
                    浅い・エンタメ（占術の深さ）
```

### fortune の狙う空白地帯

```
「本格占術 × リアル融合」の右上象限は完全な空白。
- 占いの館は「本格的だが対面・高価格・デジタル弱い」
- LINE占い/ウラナは「手軽だが浅い・デジタル完結」

→ fortune は「本格 × AI手軽 × リアル参詣」で唯一のポジション
```

## 7. ブランドの「勝ち筋」戦略

### 短期（Phase 1）: ニッチで信頼を築く


```
狙い: AIチャット + 神社連携 + コミュニティで一般層へ

- サブスク・オークション本格化
KPI: MAU 100万、課金率30%
```

### 長期（Phase 3+）: カテゴリーリーダーへ

```
狙い: 「和の占い × テック」の代名詞に
施策:
- 神社との排他的パートナーシップ
```
