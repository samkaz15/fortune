## Executive Summary

GM3の調査で判明した「非効果的なテキスト」と「効果的なテキスト」の差(CTR +40〜50%)を踏まえ、
3つの課金導線(サブスク／追加クレジット／オークション)のコピーを、
「機能の説明」ではなく「体験の予告」として書き直す。

トーンはGPT1で確定した**ツクヨミの人格**(断定しない・煽らない・共感的)を厳守しつつ、
GM3が明らかにした購買心理の原則(フリクション低減・アンカリング・購入後体験)を反映する。

**絶対原則(GPT1 NG表現との整合)**：
「今課金しないと運気が下がります」のような不安煽り型コピーは、GPT1で明示的に禁止されている。
本ドキュメントのコピーはすべてこの制約の範囲内で設計している。

## 1. コピー設計原則

### GM3知見の反映

| GM3の知見 | 本コピーでの反映 |
|---|---|
| 「ロック解除には¥300必要です」は非効果的、「続きが見たくなる」文言が効果的 | 金額を主語にせず、結果への好奇心を主語にする |
| フリクション最大の要因は「クレジットカード入力」「複数ステップ」 | Stripe Checkout(Apple Pay/Google Pay対応)への導線を1タップで開始できる文言にする |
| 3択(基本/スタンダード/プレミアム)がスタンダードの購買率を+50%押し上げる | `/plans`のタブUIで「サブスク」を先頭・強調表示し、アンカーの役割を持たせる(実装済みのRecommendedBadge相当) |
| 購入後体験の即時性がリピート購買率を+20%押し上げる | 決済完了後は`/plans/complete`で即座に結果へ戻す設計(CL10で実装済み)に合わせ、コピーも「待たせない」トーンにする |

### ツクヨミボイスのルール(GPT1準拠)

```
✅ 「〜してみませんか」「〜かもしれませんね」等、選択の余地を残す
✅ 金額は隠さず明記する(フリクション低減。「わかりにくい価格」はGM3でも逆効果と判明)
❌ 「今すぐ」「今だけ」等の緊急性を煽る表現は使わない(ツクヨミの落ち着いたトーンに反する)
❌ 「不運になります」「損します」等のFOMO(喪失訴求)は使わない(GPT1 NG表現)
```

GM3は一般的なCTR最大化施策として「FOMO」「限定感の煽り」を効果的と分析しているが、
ツクヨミのキャラクター性(不安を煽らない)と衝突するため、**採用しない**。
差別化軸(GPT2)である「信頼できる関係」を優先する。

## 2. サブスクプラン コピー

### プラン提示部分

```
【見出し】
ずっと、そばに。

【本文】
初月 500円

2ヶ月目から 月980円

診断も、相談も、回数を気にせず。
毎日のあなたに、ツクヨミがそばにいます。

【内訳リスト】
・診断回数の上限なし
・結果はいつでも全文読める
・週替わりオークションへの参加権
```

### CTA文言

```
プライマリ: 「このまま始める」
（「今すぐ登録」のような緊急性訴求は避け、静かな後押しにする）
```

### 解約導線に添えるコピー（マイページ/退会フロー用・DEVELOPMENT_RULES的な誠実さの担保）

```
「いつでも、ここでやめられます。」
```
→ サブスクの説明文の下に小さく添える。安心感を先に渡すことで、
かえって解約率が下がる(GM3「返金保証」効果の応用、ただし金銭訴求ではなく安心訴求に変換)。

## 3. 追加クレジット コピー

現状の要件（1日5回の無料枠を使い切った直後に提示）を踏まえ、
「結果への好奇心の続き」として提示する。ペイウォール文言のGM3ベストプラクティスを直接反映。

### 結果画面ロック時（`/result/:id`）に表示するコピー

```
【見出し】
今日のあなたには、まだ続きがあります。

【本文】
ここから先は、もう少し深く、あなたの流れを読み解いた内容です。
```

### `/plans`のクレジットタブ

```
【見出し】
もう少しだけ、話したい日に。

【価格提示】
300円で、5回分

【補足】
今日の無料分を使い切っても、その場で続けられます。
```

### CTA文言

```
プライマリ: 「続きを聞く」
（「クレジットを購入」ではなく、チャットの続きという体験を主語にする）
```

## 4. オークション（面談チケット） コピー

GM4(オークション型課金モデル調査)・GM3の社会的証明/希少性の知見のうち、
**ツクヨミの人格と矛盾しない範囲**(希少性は事実として伝えるが、煽らない)で採用する。

### 一覧画面 見出し

```
【見出し】
週にいちど、ツクヨミと直接話せる時間。

【補足】
1つの面談枠を、24時間の入札でお分けしています。
```

### 個別チケットカード内

```
【現在価格ラベル】
今の入札額

【入札件数の見せ方】
「◯件の入札」（社会的証明。数値のみ淡々と提示し、煽り文言は付けない）

【残り時間ラベル】
「残り◯時間◯分」（事実の提示に留め、「あと少し！」等の煽りは付けない）
```

### 入札後の確認メッセージ

```
「入札を受け取りました。ここからの流れは、そっと見守っていてくださいね。」
```

### 落札後の案内（Phase2の面談予約完了画面で使用予定）

```
「おめでとうございます。あなたの時間を、楽しみにしています。」
```
