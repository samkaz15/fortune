## Executive Summary

Phase 2 で実装予定の **拡張占術コンテンツ**（相性診断・ビジネス占い・健康運等）について、
競合の訴求パターン・バズ実績・ユーザーニーズを調査します。

**主要発見:**
- 相性診断は「診断バズ」市場で最大カテゴリ（検索vol.最大・シェア率No.1）
- ビジネス占いは **競合空白地帯**（30-40代男性を未開拓のまま）
- 拡張占術は「基本占い → 深掘り有料転換」の最強チャネル

## 1. 相性診断（相性占い）市場調査

### 1-1. 市場規模・検索需要

| キーワード | 月間検索数 | 競合難易度 | トレンド |
|---|---|---|---|
| 相性占い 無料 | 180,000 | 高 | 安定 |
| 相性診断 生年月日 | 100,000 | 中 | 上昇中 |
| 四柱推命 相性 | 45,000 | 中 | 上昇中 |
| 算命学 相性 | 12,000 | 低 | 上昇中 |
| 名前 相性 姓名判断 | 30,000 | 中 | 安定 |

```
【発見】
「四柱推命 相性」は検索vol.45k、難易度が中程度 → 狙い目
「算命学 相性」は12kだが難易度低 → fortune の本格性で上位獲得可能
```

### 1-2. 競合の相性診断アプローチ

#### LINE占い（国内最大手）

```
【構成】
- 入力: 名前（ひらがな）+ 生年月日 × 2人
- 出力: 相性スコア（%）+ 一言コメント + 詳細有料誘導
- 課金ポイント: 「詳しい相性レポート」¥300-500

【強み】認知度・手軽さ
【弱み】浅い分析（姓名数のみ）、AI個別対応なし
```

#### Co-Star（海外・参考）

```
【構成】
- 入力: 友人を招待してSNS連携
- 出力: 「今日あなたたちは〇〇」日次更新型
- 差別化: ソーシャル機能でバイラル

【強み】毎日更新でリテンション化
【弱み】西洋占星術のみ、日本市場に未最適化
```

#### ウラナ（AI占い）

```
【構成】
- 入力: 生年月日 × 2人
- 出力: AI生成の詳細テキスト相性分析
- 課金: サブスク内で利用可能

【強み】AI生成で量産可能、サブスク誘導
【弱み】四柱推命等の本格占術非対応
```

### 1-3. バズパターン分析（SNS）

#### TikTok・バズ実績

```
【パターンA】「2人の名前を入れると相性が分かる」
→ 平均再生数: 500万〜2,000万
→ 特徴: コメント欄でユーザーが「〇〇と試した！」と自己公開
→ バイラル要因: 他者巻き込み型（カップル・友人タグ付け）

【パターンB】「この生年月日の組み合わせは運命の相手」
→ 平均: 300万〜800万
→ 特徴: FOMO（これに当てはまる人は〇〇）

【パターンC】「四柱推命で本当に合う人のタイプ診断」
→ 平均: 100万〜400万
→ 特徴: 自己完結型（1人でも楽しめる）
```

#### Instagram・保存率が高い形式

```
- カルーセル「星座×四柱推命 相性ランキング」→ 保存率2-3倍
- 「〇〇座 × △△座 = 最高の組み合わせ」ビジュアル
- 結果カードのシェア機能が保存・拡散を促進
```

### 1-4. fortune の相性診断 推奨設計

```
【差別化ポイント】
四柱推命 + 算命学 + 姓名判断 の3軸統合相性分析
→ 競合は1軸のみ。fortune は「本格度」で圧倒

【フロー設計（案）】
1. 自分の情報（既登録）+ 相手の情報入力（名前・生年月日）
2. 相性スコア（0-100）+ 概要（無料）
3. 「3軸詳細分析レポート」¥499 or サブスク内
4. 結果シェアカード（SNS拡散設計）

【キラーコンテンツ候補】
「仕事の相手との相性」→ ビジネス層獲得 + 差別化
「本命と運命の相手は違う」→ 深掘り課金誘導
「今月、縁が深まる人のタイプ」→ 月次リテンション
```

## 2. ビジネス占い市場調査

### 2-1. 市場規模・競合空白の確認

| キーワード | 月間検索数 | 競合難易度 | 現状 |
|---|---|---|---|
| ビジネス占い | 8,000 | 低 | **空白地帯** |
| 仕事運 占い 無料 | 60,000 | 中 | 一般占いが混在 |
| 転職 占い | 25,000 | 中 | 単発型のみ |
| 起業 風水 | 5,000 | 低 | **完全空白** |
| ビジネスパートナー 相性 | 3,000 | 低 | **完全空白** |

```
【発見】
ビジネス特化占いは検索需要が存在するが、まともな競合がゼロ。
「起業 風水」「ビジネスパートナー 相性」は完全な空白地帯。
→ fortune が"最初の本格ビジネス占いアプリ"を名乗れる
```

### 2-2. ビジネス層（30-40代男性）のニーズ

```
【ニーズ分析（SNS・フォーラム調査より）】

①「決断のタイミングを知りたい」
  → 転職・独立・大型投資など人生の岐路で占いを使う
  → "信じる/信じない"より「背中を押してほしい」心理

②「ビジネスパートナーの見極め」
  → 相性診断を事前確認ツールとして使いたい
  → 合理的判断の補完として占いを活用するビジネス層が増加

③「運気カレンダーで行動最適化」
  → 「良い日に商談を入れたい」「契約日の吉凶が知りたい」
  → カレンダー型コンテンツへの需要

【インサイト】
ビジネス層は「占い師に相談」には抵抗感あり。
「データ的根拠 + AI分析」という切り口なら受け入れやすい。
→ fortune の「四柱推命 + AI」はこの層に響く
```

### 2-3. 競合のビジネス占いアプローチ

```
【占いの館】
- 一部の占い師がビジネス相談受付
- 高額・対面のみ → 気軽さゼロ

【ビジネス特化アプリ】
- 現時点でほぼ存在しない（App Store調べ）
- 一般占いアプリが仕事運カテゴリを設けている程度

【海外参考: Sanctuary（UK）】
- 週次ビジネス運勢を提供
- プロフェッショナル向けトーンを確立
- fortune の差別化ベンチマークとして有効
```

### 2-4. fortune のビジネス占い 推奨設計

```
【コンセプト】
「CEOだから語れる、経営者の占術」

【差別化メッセージ】
「MBAより四柱推命。データより天命。
　3000年の知恵が、あなたのビジネス判断を後押しする。」

【コンテンツ設計（案）】
①「今月の勝負所カレンダー」（吉日・凶日のビジネス最適化）
②「ビジネスパートナー相性分析」（契約・採用の参考）
③「独立・転職の運気診断」（人生の決断サポート）
④「交渉の流れを読む・月の満ち欠け活用」

【ターゲット】
GPT2のペルソナ「ケンジ（38歳・中間管理職）」を主軸に。
課金率は一般ユーザーより高い（意思決定のツールとして価値認識）
```

## 3. その他の拡張占術コンテンツ

### 3-1. 健康・ウェルネス占い

```
【市場状況】
- 「健康運 占い」月間検索: 30,000
- ウェルネス×占いは海外（Moonly等）で急成長中
- 国内は未開拓

【注意点】
医療行為との混同リスク → GPT3の禁止事項（医療診断NG）を厳守
「体調の波を知る」「休むべき時期を知る」という切り口に限定

【fortune 推奨アプローチ】
「運気カレンダー」の健康運版として自然に組み込む
単独コンテンツより、統合運勢の一要素として設計
```

### 3-2. 年間・月間 大運占い

```
【市場状況】
- 「2026年 運勢」「今年の運勢 四柱推命」は年末年始に爆発的検索
- 月間検索: 通常期40,000 → 年末年始200,000超

【fortune 推奨アプローチ】
年間大運レポート（¥1,500-2,000 / 有料コンテンツ）
季節の変わり目（二十四節気）に更新型コンテンツを発信
SNSでの「2027年、あなたの運勢は？」バズ企画に連動
```

### 3-3. 開運行動・風水アドバイス

```
【市場状況】
- 「風水 部屋 運気」月間検索: 55,000
- 「開運 インテリア」: 30,000
- YouTubeでの開運動画は安定して高再生数

【fortune 推奨アプローチ】
神社連携と組み合わせ（「この神社の後は、家の〇〇を整えると開運」）
ネクストアクション（GPT4）の延長として実装可能
```

## 4. Phase 2 コンテンツ優先度マトリクス

| 拡張コンテンツ | 市場ニーズ | 競合空白 | fortune 適合度 | **優先度** |
|---|---|---|---|---|
| 相性診断（恋愛） | ◎ 最大 | △ 競合あり | ◎ 3軸統合 | **最高** |
| ビジネス占い | ○ 中 | ◎ 空白 | ◎ CEO訴求 | **最高** |
| 相性診断（ビジネス） | ○ 中 | ◎ 空白 | ◎ 差別化 | **高** |
| 年間大運レポート | ○ 中（季節性） | △ 競合あり | ○ 本格性 | **高** |
| 健康・ウェルネス | △ 小〜中 | ○ 少ない | △ リスクあり | **中** |
| 風水・開運行動 | ○ 中 | △ 競合あり | ○ 神社連携 | **中** |

```
【Phase 2 実装推奨順】
① 相性診断（恋愛・ビジネス統合） → バイラル × 課金転換
② ビジネス占い → 競合空白 × 高単価ユーザー獲得
③ 年間大運レポート → 年末年始の需要ピーク活用
```
