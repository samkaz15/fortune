## 1. TikTok での占い・神社コンテンツのバズパターン

### A. 「あるある系」占い動画

**バズ事例**: 「あなたの血液型別・性格診断」シリーズ

```
【動画フォーマット】
- 長さ: 15-30秒
- 冒頭: 「これ当たりすぎて怖い」（テロップ）
- 構成: 4タイプを素早く紹介
- BGM: アップテンポなトレンド曲
- エンディング: 「あなたはどれ？コメントで教えて」

【バズのポイント】
✅ 見終わって「自分はこれだ」と確認したくなる
✅ 友人に「これ見て」とシェア → 流行拡大
✅ 多くの人に当てはまるように設計

【再生数・エンゲージメント】
- 平均再生数: 500万回+
- シェア率: 20-30%
- 関連動画: 3-5個の派生形が流行
```

**fortune への適用**:
```
「あなたの四柱推命タイプ」動画シリーズ
例: 「5つのタイプで占い」→ 
・木のタイプ: 活動的
・火のタイプ: 情熱的
・土のタイプ: 安定志向
・金のタイプ: 分析的
・水のタイプ: 包容力

効果: 1つの動画で 500万回再生を狙う
```

### B. 「占い結果が当たりすぎた」ショック系

**バズ事例**: 「生年月日から隠された性格が判明」

```
【動画フォーマット】
- 冒頭: 「信じるか信じないかはあなた次第」
- ストーリー: ユーザーの生年月日を入力 → 結果表示
- ビジュアル: 炎のエフェクト、光のエフェクト
- エンディング: 「怖い…」「これ本当？」のコメント促進

【バズのポイント】
✅ 「精度が高い」というリアルな反応に見える
✅ コメント欄が賑わう = アルゴリズム優遇
✅ 「友人に試してみて」とシェア

【再生数・エンゲージメント】
- 平均再生数: 1000万回+
- コメント数: 5万件+
- 保存数（後で見る）: 高い
```

**fortune への適用**:
```
「あなたの隠された才能が判明」シリーズ
入力: 名前 + 生年月日
出力: 
  - 隠された才能 3つ
  - 注意すべき弱み 1つ
  - 相性の良い人物タイプ

テクニック: 
  ✅ 分岐の多い結果（全部で 81 パターン等）
  ✅ 「すべてに当てはまってしまう」現象（バーナム効果）
```

## 2. Instagram での占い・診断コンテンツ

### A. 「Reels」での診断動画（短編）

**バズ事例**: 「3秒で分かる性格診断」

```
【フォーマット】
- 長さ: 15-60秒
- 構成: 
  1. 質問 (冒頭 3 秒)
  2. 選択肢表示（ビジュアル重視）
  3. 結果（サプライズ性）
  4. CTA（「もっと診断したい」→ プロフィール link）

【バズのポイント】
✅ Reels は Instagram のトップに配置
✅ シェア→友人も試す→フォロワー増
✅ CTA で Web 誘導が可能
✅ ストーリーズ保存で「後で見る」機能

【エンゲージメント】
- 平均再生数: 50-200万回
- リンククリック率: 5-15%
- フォロワー増加: 週 500-2000人
```

**fortune への適用**:
```
Instagram Reels の毎日更新
例: 「今週のあなたの運勢」(毎月曜投稿)
  - 曜日別に異なる診断
  - 月~日で 7 パターン
  - 自分の曜日を選んでシェア

効果:
  ✅ 毎日のユーザー訪問
  ✅ Web へのトラフィック
```

### B. 「診断結果をストーリーズでシェア」キャンペーン

**バズ事例**: 「あなたの魔女タイプは？」

```
【メカニズム】
1. Web で診断を実施
2. 結果をインスタ画像にして提供
3. ユーザーが IG ストーリーズで共有
4. 「僕はどのタイプ？」で友人も参加
5. タグ #○○診断 で集約

【バズのポイント】
✅ UGC（ユーザー生成コンテンツ）の拡大
✅ ハッシュタグで検索流入
✅ インスタのアルゴリズムが UGC を優遇

【成功事例のエンゲージメント】
- ハッシュタグ使用数: 100万件+
- バイラル期間: 2-4週間
- Web 流入: 月 500万クリック
```

**fortune への適用**:
```
「fortune の診断結果」をインスタシェア可能に
- 毎回異なるビジュアル（月別、季節別）
- ユーザーの名前を入れた「あなただけの結果」
- シェア→友人参加→ラッキーカラー増加

ハッシュタグ: #fortune占い #あなたの運気
```

## 3. YouTube Shorts / TikTok での ASMR コンテンツ

### A. 「占い×ASMR」のハイブリッド

**バズ事例**: 「瞑想系占いガイド」

```
【フォーマット】
- 長さ: 10-15分
- 構成:
  1. 瞑想導入（3分）
  2. 占い分析（8分）
  3. 瞑想クロージング（2分）
- 音声: 専門家の低い声（ASMR効果）
- BGM: 自然音（鳥、水音）

【ASMR の効果】
✅ リラックス効果 → 睡眠補助コンテンツとして活用
✅ 「聴きながら寝る」ユーザーが増加
✅ 高い再生時間（ユーザーが最後まで聴く）

【エンゲージメント】
- 再生数: 500万-1000万回
- 平均再生時間: 60-80%（通常は 30-50%）
- コメント: 「毎晩聴いてます」の感謝コメント多数
```

**fortune への適用**:
```
「毎晩の瞑想占いガイド」YouTube チャンネル
- 「今夜のあなたの運勢」を瞑想しながら診断
- CEO の声による「個人的なメッセージ」
- 毎晩更新（就寝前の習慣化）

効果:
  ✅ 高い再生時間→YouTube アルゴリズムに優遇
  ✅ 毎晩訪問→継続率向上
  ✅ コメント→コミュニティ形成
```

## 4. TikTok/Instagram での「神社チャレンジ」

### A. バイラルチャレンジの設計

**バズ事例**: 「この神社に行ったら運気UP」チャレンジ

```
【メカニズム】
1. fortune が「推奨神社」を指定
2. ユーザーが実際に参詣
3. 神社での動画撮影（#fortuneチャレンジ）
4. TikTok/Instagram に投稿
5. 入選者に特典（App 内クレジット等）

【バズのポイント】
✅ 実際の行動が伴う（観光効果）
✅ UGC が多量に発生
✅ 神社側も喜ぶ（参詣者増加）

【成功期待値】
- 参加動画: 1000-5000件
- 利用神社: 月 3-5社
- App へのフロー: 動画視聴者の 10-20%
```

**fortune への具体案**:
```
「fortune 参詣チャレンジ」
・毎月 3 社の「ラッキー神社」を指定
・ユーザーが参詣して動画撮影
・App で「参詣完了バッジ」をGET
・10 社参詣で「CEO オークション権」獲得

タグ: #fortune参詣チャレンジ #運気UP
```
