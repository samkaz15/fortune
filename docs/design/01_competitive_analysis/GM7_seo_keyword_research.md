## 4. 神社・パワースポット コンテンツ戦略

### A. 地域別 SEO（Local SEO）

```
【ターゲット】
都市別: 東京, 大阪, 京都, 福岡等

【記事例】
「東京のパワースポット 神社 TOP 10」
  → Google Map 連携で地図表示
  → 「近い順」検索に対応

「四柱推命で分かる あなたにぴったりな神社 (東京)」
  → 「東京 パワースポット」検索者へ
  → 占いと実地参詣を連携

【期待効果】
- 地域キーワード × 占い キーワードの複合
- 検索 + Google Map での流入
- 神社への観光客増加 = 社会的評価向上
```

### B. CEO の「参詣レビュー」コンテンツ

```
【フォーマット】
「CEO が行った 〇〇神社 参詣レビュー」

各記事:
  - 神社の歴史・特性
  - 参詣方法
  - 「このタイプの人にぴったり」という占い分析
  - 実際に感じたエネルギー/雰囲気

【SEO 効果】
- Unique Content（他サイトにない一次情報）
- 長尺記事（SEO的に有利）
- 内部リンク（fortune App へ）
- Backlink 候補（神社公式サイトからのリンク）

【期待トラフィック】
年 50 記事 × 月 1000 search impression = 月 50000 PV
```

## 5. fortune App への自然な誘導戦略

```

### B. 「有料コンテンツ」への誘導

```
【記事】「2024年 あなたの運勢完全ガイド」
  内容: 基本的な運勢分析 (無料)

    ↓
  App の「詳細解釈」（有料 $4.99）へ

【効果】
検索ユーザー → 無料コンテンツ消費 → 有料へのアップグレード
