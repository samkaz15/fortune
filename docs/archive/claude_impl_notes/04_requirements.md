# CL4: 要件定義（改訂反映・要約）

## 機能要件（実装済み範囲）

- 占いチャット（人間関係/自分のこと/ビジネス/相性/今日の占い）を1API・1画面で提供する。
- 無料枠：1日5回（質問単位で合算）。超過後は追加クレジット(300円/5回)またはサブスクで継続可能。
- サブスク：初月500円 → 2ヶ月目以降980円（Stripe Coupon等で初月割引を表現）。
- オークション：週1回、開始価格1,000円、24時間限定、楽観ロックで同時入札を制御。
- 診断結果は未ログインでも閲覧可能（要約まで）。全文はサブスク会員 or 単発購入者のみ。
- 天気（低気圧等）を任意で診断コンテキストに反映する。

## 非機能要件

- 二重生成防止：Redis分散ロックでAI呼び出しの多重実行を防ぐ。
- PII分離：`UserProfile` に集約し、他ドメインテーブルから直接参照しない。
- 決済情報：カード番号等は自社DBに保持せずStripeに委譲。
