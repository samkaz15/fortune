# 総合占術アドバイザー レビュー依頼パック(第1回)

作成日: 2026-07-12 / 作成: 開発チーム / 対象: 占術計算エンジンの叩き台(CEO1監修シート D-2・D-6・D-7・D-9)

本パックは「仕様をゼロから書いていただく」ためのものではなく、**実装済みのロジックとデータセットに対する修正・承認(セカンドオピニオン)**をお願いするものです。各項目に「確認方法」を付けています。

---

## A. 最優先: 日柱基準日の修正の追認(D-9)

旧実装は「1984年2月2日=甲子日」を基準にしていましたが、独立した暦計算実装2系統との突き合わせで**1984年2月2日は丙寅日**であり、正しい甲子日は**1984年1月31日**と判明したため修正済みです(全ユーザーの日柱が2日ズレていたことになります)。

**お願い**: お手元の万年暦(市販2冊以上)で以下の抜き取り確認をお願いします。
- 1984年1月31日 = 甲子 / 1984年2月2日 = 丙寅 / 2000年1月1日 = 戊午 / 1924年2月15日 = 甲子

## B. 節入りテーブルの抜き取り照合(D-6)

1900〜2100年の十二節(小寒〜大雪)の節入り日時テーブルを整備しました(`setsuiri-1900-2100.json`)。元データは中国標準時のため**+1時間して日本標準時に変換済み**です。この変換により**節入りの日付が繰り上がるケースが201年間で101件**あります(未変換だと日本基準の命式が1日ズレる要注意ポイント)。

**お願い**: 運命暦・専門書と、任意の5〜10年分(特にご自身がよく参照される年代)の立春・啓蟄などの日時を照合してください。分単位の小差は許容、**日付のズレがあれば要報告**です。

## C. 夜子時の流派確認(D-7)

23時台生まれの扱いは流派が分かれるため、以下を既定としています(切替オプション実装済み):
- **既定(夜子時説)**: 日柱は当日のまま、時干のみ翌日の日干から五鼠遁で起こす
- オプション(換日説): 23時以降は日柱ごと翌日に切り替える

**お願い**: 高尾学館系の実務でどちらを標準とすべきかご指定ください。あわせて、出生時間が不明な場合の扱い(現状: 時柱なしの三柱として鑑定)もご確認ください。

## D. 姓名判断の画数データ検品(D-2)

新字体→旧字体変換(419字)+画数辞書(3,199字)を整備しました。字典間で画数が揺れた文字は以下のルールで暫定処理し、**全件を `seimei-review.json` に列挙しています**:
1. 筆順データ(KanjiVG)と字典(KANJIDIC)が不一致 → 筆画数を採用(51字)
2. 旧字体の画数データが欠損・康熙式混入疑い → 新字体の画数へフォールバック(42字。例: 郞)

**お願い**:
- `seimei-review.json` の93字について、熊崎式で用いるべき画数のご指定
- 康熙字典基準の部首画数補正(氵=水4画、艹=艸6画、辶=辵7画、阝=邑7画/阜8画 等)を**適用すべき文字と画数の一覧**のご提供(`radical-adjustments.json` に投入すれば辞書より最優先で反映されます)
- かな画数表 `kana-strokes.json`(165字)の全値確認
- 霊数の算式確認: 現実装は「一字姓→天格+1 / 一字名→地格+1 / 総格は霊数を含まない / 外格=天格+地格−人格」

## E. 1〜81画吉凶表のご提供(D-4)

`kikkyou-table.json` に投入枠を用意済みです。各画数に `{rank: 大吉|吉|半吉|凶|大凶, score: 0-100}` の形式でご提供ください。ご提供までは暫定採点で稼働し、投入した瞬間から正式判定に切り替わります。

## F. 検証ケースのご作成(D-0b)

以下の形式で各10件お願いします。いただき次第、自動テストに固定し、以後の改修の合格条件にします。
- 姓名判断: 氏名 → 五格の画数と吉凶
- 四柱推命: 生年月日時 → 四柱(特に立春前後・節入り当日・23時台生まれを各1件以上)
