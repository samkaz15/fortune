作成日: 2026-07-11 / 根拠: `docs/marketing/01〜10`(正式な設計思想として採用) / 対象: 既存プロダクトの改善(新規開発ではない)

## 1. 現状分析

**強み(マーケレポートで裏付けられた資産):**
- 日次起動動機の核「今日の運勢」が実装済み(継続率の土台。業界Day30平均29%に対し、参加型日次コンテンツは底上げ効果が実証されている — 01/02章)
- 「無料診断→本格相談(トークション)」の二段導線は業界標準の成功構造と一致(01章)
- 仕事×算命学診断は競合が薄いブルーオーシャン(03章D)
- 運営者情報・プライバシーポリシー整備済み = AI占い懐疑層への信頼性訴求材料(01章)
- 5占術エンジン(算命学・四柱推命・九星気学・姓名判断・ホロスコープ)+スコアv3(日次変動)実装済み

**弱み:**
- 占い結果に占術上の根拠(天中殺・月破・中宮等)が明示されず「AI文章っぽさ」が残る → 信頼性懸念(01章のコールドリーディング疑念)に直結
- 七夕世界観が背景(天の川)のみで、コピー・体験に浸透していない
- FAQ・構造化データ(FAQPage)・llms.txt が未整備 → SEO/AEO/LLMOの検索資産が弱い
- ストリーク(連続利用)は計算実装済みだが、今日の運勢画面に露出していない(02章の施策2が半実装)
- シェアコピーが汎用的(Marketing-009未着手)

## 3. UI/UX改善案(根拠付き)

1. **占術根拠ブロック**(要件6): 今日の運勢・自分のことに「今日の星回り」欄を新設し、天中殺・月破・中宮・命式を明示 — 根拠: 01章の信頼性懸念対策、03章EEAT
2. **七夕世界観の強化**(要件3): スコア欄を「今日の星回り」文脈へ、シェアを「短冊に書く」メタファーへ、ローディング・締めコピーに天の川・織姫彦星・願いの語彙 — 根拠: ブランド体験の差別化(01章: 海外勢はビジュアル重視UI)
3. **ストリーク表示**(継続率): 「◯日連続でチェック中」を今日の運勢へ — 根拠: 02章リテンション施策2、北極星指標(週3回以上開くユーザー)への直接寄与
4. **トークション**(要件2): 入札件数のライブ表示(盛り上がりの可視化)+安心コピー — 根拠: 02章「課金の瞬間の体験設計」
5. **オンボーディング**: 初回入力画面に「初回のみ・次からワンタップ」を明示(実装済み)+FAQへの導線 — 根拠: 02章FTUE(初回体験が良いと継続率+300%)
