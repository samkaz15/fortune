## Executive Summary

fortune の中核体験は「占って終わり」ではなく、
**"今日、あなたが取るべき具体的な一歩"を示すこと**です（GM5 の差別化軸3）。

GM3 の調査で、「具体的で実行可能な次の一歩」の提示が
継続率を大きく左右することが判明しました。

本ドキュメントは、その「ネクストアクション」を
**PC2行 / モバイル5行に収まる簡潔さ**で、かつツクヨミらしい
温かさで生成するためのテンプレートを定義します。

---

## 1. 設計原則

### 4つの原則（GM3 の知見を反映）

```
1. 具体的（Specific）
   ❌「運気を上げましょう」
   ✅「朝、窓を開けて深呼吸を3回」

2. 実行可能（Achievable）
   ❌「人生を変える大きな決断を」
   ✅「気になっていた人に、一言メッセージを」

3. 前向き（Positive）
   常にポジティブな行動。ネガティブ回避型でも希望を持たせる

4. 簡潔（Concise）
   モバイル5行以内。1アクション = 1行 + 補足少々
```

### 文字数制約

| 表示環境 | 制約 | 内訳 |
|---|---|---|
| **PC** | 2行 | 見出し1行 + アクション要約1行 |
| **モバイル** | 5行 | 見出し1行 + アクション3つ + 締め1行 |

---

## 2. 基本テンプレート構造

### モバイル版（5行）

```
🌙 今日のあなたへ                    ← 見出し（1行）

1. [アクション1]                     ← 具体行動（1行）
2. [アクション2]                     ← 具体行動（1行）
3. [アクション3]                     ← 具体行動（1行）

大丈夫、あなたのペースで。            ← 締め（1行）
```

### PC版（2行）

```
🌙 今日のキーアクション: [最重要アクション1つ]
　 [補足: なぜ今日それが良いのか、一言]
```

## 3. カテゴリ別バリエーション

運気スコア・占術結果に応じて、アクションの方向性を変えます。

### A. 高運気日（スコア 80-100）——「攻めの日」

```
🌙 今日は追い風の一日

1. 前から迷っていたことに、一歩踏み出して
2. 大切な人に、感謝を言葉にして伝えて
3. 新しい出会いの場に、顔を出してみて

チャンスの日。あなたなら、大丈夫。
```

### B. 中運気日（スコア 50-79）——「整える日」

```
🌙 今日は穏やかに整える一日

1. デスクや部屋を、少しだけ片付けて
2. いつもより丁寧に、食事を味わって
3. 気になっていた連絡を、一つ返して

焦らず、あなたのペースで。
```

### C. 低運気日（スコア 20-49）——「守りの日」

```
🌙 今日は静かに過ごしたい一日

1. 大きな決断は、明日以降にそっと保留
2. 温かい飲み物で、ひと息ついて
3. 早めに休んで、自分をいたわって

"注意"は"不運"じゃない。備えれば大丈夫。
```

### D. カテゴリ特化（仕事運）

```
🌙 今日の仕事運を活かすには

1. 午前中に、一番大事なタスクを片付けて
2. 迷ったら、信頼できる人に相談を
3. 小さな成果も、しっかり記録して

あなたの努力は、ちゃんと実を結びます。
```

### E. カテゴリ特化（恋愛運）

```
🌙 今日の恋愛運のヒント

1. 気になる人に、さりげない一言を
2. 自分を飾らず、素直な気持ちで
3. 相手の話を、ゆっくり聞いてみて

縁は、焦らないほうがうまくいくもの。
```

## 4. LLM生成用プロンプト（テンプレート化）

Claude/OpenAI が動的にネクストアクションを生成する際のプロンプト。

```markdown
# 役割
あなたは占いアプリ「fortune」のキャラクター「ツクヨミ」として、
ユーザーへの「今日のネクストアクション」を生成します。

# 入力
- 運気スコア: {score}（0-100）
- 占術タイプ: {fortune_type}（総合/仕事/恋愛/健康）
- ラッキーカラー: {lucky_color}
- ユーザー属性: {user_context}

# 出力ルール
- モバイル5行以内（見出し1 + アクション3 + 締め1）
- 各アクションは「具体的・実行可能・前向き・簡潔」に
- スコアに応じてトーン調整:
  - 80以上: 攻めの提案
  - 50-79: 整える提案
  - 49以下: 守りの提案（но希望を必ず残す）
- ツクヨミの口調（敬体・余白・温かさ）を守る
- 締めは必ず前向きな一言で終える

# 禁止
- 医療・投資・断定的不幸予言（GPT3の禁止事項に準拠）
- 抽象的すぎるアクション（「頑張る」等）

# 出力形式
🌙 [見出し]

1. [アクション1]
2. [アクション2]
3. [アクション3]

[前向きな締めの一言]
```
